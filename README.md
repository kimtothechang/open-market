# Open Market

## **🌏 목적**

---

- 다중 판매자 기반의 쇼핑몰 웹 어플리케이션 제작

<br/>

## **🛠 기술**

---

- React - SPA방식의 최소 렌더링
- Recoil - 상태관리
- Emotion - CSS in JS를 통한 컴포넌트화
- yarn berry - 가볍고 빠른 의존성 관리
- Prettier & ESLint - 코드의 통일성과 컨벤션 유지
- Vercel - 배포

<br/>

## **⚙️ 역할**

---

- 프로젝트 설계 및 계획
- 프로젝트 환경 설정 및 기술 스택 적용
- 전 기능 책임 개발

<br/>

## **⚡️ 기능**

---

### 1. 공통

- 로그인 및 회원가입 기능
- 메인 배너 및 상품 리스트 제공

### 2. 구매자

- 장바구니 기능
- 상품 주문 및 결제 기능

### 3. 판매자

- 상품 등록 기능
- 판매자 센터(대시보드) 기능
